## How to Run Locally

```bash
git clone https://github.com/Hiromu-USHIHARA/ReactBlackJack.git
cd ReactBlackJack

npm install
npm start
```

Then open `http://localhost:3000` in your browser.

## Features

- Stylish UI using Mantine
- Dark mode toggle (top-right)
